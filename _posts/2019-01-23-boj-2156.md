[2156 : 포도주 시식](https://www.acmicpc.net/problem/2156)  

## 풀이

.s

## 코드

```cpp
int max(int a, int b) { return a > b ? a : b; }
n, a, b, c, A, B, C, w;
main() {
	for (scanf("%d", &n); n--;) {
		scanf("%d", &w);
		C = c, B = b, A = a;
		c = B + w;
		b = A + w;
		a = max(max(A, B), C);
	}
	printf("%d", max(max(a, b), c));
}
```
